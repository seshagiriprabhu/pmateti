2 Alternate Android App Dev Environments

2.1 App Inventor

  1. http://appinventor.mit.edu/

2.2 Cordova

"Apache Cordova is a free and open source framework that allows you to create mobile apps using standardized web APIs. You can create apps that work on iOS, Andriod, BlackBerry, Windows, Ubuntu and other phone based operating systems. You write code once and run on selected mobile platforms with little or no change at all. PhoneGap uses Apache Cordova."

2.3 PhoneGap

  1. Open-source mobile (Android + iOS) development framework
  2. HTML5, CSS3, and JavaScript
  3. More than one platform, without having to re-implement with each platform's language and tool set.
  4. Android Shell Tool Guide and iOS Shell Tool Guide.
  5. http://phonegap.com

2.4 jQuery Mobile

  1. Open-source mobile development framework http://jquerymobile.com/
  2. "jQuery Mobile is a HTML5-based user interface system designed to make responsive web sites and apps that are accessible on all smartphone, tablet and desktop devices."

2.5 Worklight (IBM)

  1. Not open source. Price: ~$170K
  2. http://www.ibm.com/software/products/en/worklight

2.6 Porting Existing Applications to Android

  1. Porting C, C++, C# and other native code
  2. Android NDK
  3. vs-android couples with MS Visual Studio
