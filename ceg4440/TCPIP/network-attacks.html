2 Network Attacks using Android

2.1 DroidSheep, Network Spoofer

  1. DroidSheep can ARP-poison. The companion DroidSheep Guard monitors the Android's ARP-table. With a few tweaks, the source code builds into .apk readily. DroidSheep is no longer available on Google Play. DroidSheep was part of a BS Thesis. http://droidsheep.de/
  2. Network Spoofer can do ARP and DNS poisoning. The site has links to its .apk that can be side loaded. Building from source is slightly difficult. Needs http://jsoup.org, etc. https://www.digitalsquid.co.uk/netspoof/

2.2 Network Attacks using Android, ig-learner

  1. IGLearner app is designed to teach internals and security of Android through puzzles/ quizzes. It is available on Google Play. FOSS: https:github.com/intrepidusgroup/ig-learner. 2013
  2. Improve this! Masters' Thesis?

2.3 Network Attacks using Android, zANTI2.apk

  1. dSploit / zANTI2 can do ARP and DNS poison.
    1. An attacking APK that combines with Web surfing
    2. https://github.com/evilsocket/dsploit/downloads dSploit APK 2012
    3. https://s3.amazonaws.com/zANTI/zANTI2.apk 2015 dSploit merges with ZImperium
    4. Details: The "Download" section gives links to .apk that can be side loaded. This version is free from ads. (Some unscrupulous people are releasing FOSS on Google Play with ads inserted, benefitting them instead of the author(s) of FOSS.) The "How To Contribute" section describes how to build an apk out of the source code obtained from GitHub. Web search for a dSploit tutorial that suits you. (Google now tags this site as "unsafe". I do not think this is warranted.)
