Android Security Experiments Lab Setup

  1. We will not be using Windows (any version).
  2. Linux on x86-64 machines. Decent CPU. 8+ GB RAM. 50+ GB available HDD. Preferably running Kubuntu latest.
  3. Android Studio installed. http://developer.android.com/sdk/index.html
  4. Android SDK Stand-Alone https://developer.android.com/sdk/installing/index.html?pkg=tools
    1. Include Emulator / AVD
    2. Includes adb, fastboot
  5. Install location for the above two: /usr/local/
  6. run, as root, the following script /usr/local/android-sdk-linux/tools/android Update the packages to the latest versions. Also, install ARM 64 V8a (Android N) and ARM EABI V7a (Android 6.0.0+ versions) system images.
  7. IntelliJ Idea Community Edition installed https://www.jetbrains.com/idea/download/
  8. Eclipse latest edition installed https://www.eclipse.org/downloads/
  9. Oracle JDK 8, 7, and 6 (all three for x86-64) installed http://www.oracle.com/technetwork/java/javase/downloads/
  10. Keep the OpenJDK – do not delete
  11. Virtual Box / VMware setup
